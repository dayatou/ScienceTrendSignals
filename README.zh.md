# ScienceTrendSignals 项目说明

[English](./README.md) | [Technical README](./README-tech.md) | [技术说明（中文）](./README-tech.zh.md)

ScienceTrendSignals 是一个面向科研领域表征的分析项目。项目同时处理四个维度：

- 语义结构
- 时间演化
- 战略位置
- 地理分布

仓库将语料分析、空间预处理和 GIS 展示整合在同一流程中。

## 研究范围

项目的主要分析对象包括：

- 语料中的主题簇
- 主题与子领域之间的层级关系
- 主题数量、近期性和增长特征
- 核心、邻近、桥接和前沿等战略角色
- 作者与机构的空间集中
- 国家、城市和机构层面的时间性重心转移

## 科研意义

项目将三种通常分离处理的分析合并到一起：

- 研究主题的语义地图
- 主题成熟度与前沿潜力的战略判断
- 作者 affiliation 记录的空间地图

输出对象是领域层面的结构，而不只是文献条目的描述。

## 战略相关性

项目支持以下判断：

- 稳定核心方向
- 核心领域的邻近与支撑方向
- 连接不同知识区域的桥接方向
- 高增长或具有前沿潜力的方向
- 与增长相关的地理中心
